冰镇大西瓜

2018年05月17日 冰爽夏天化

最喜欢把西瓜一刀切两半，用勺子舀着吃，小时候觉得特威风。

每当弟弟来抢我西瓜吃的时候，我就抱着半个西瓜围着我们家院子跑，现在想起来小时候的这种场景特别温暖。

图1    Select Web文章列表

现在弟弟终于跟我在一个城市了，又可以像小时候一样，西瓜切两半，他一半我一半，拿着个大勺子，坐在马扎凳上吃西瓜。或者对着弟弟吼一句，快过来，抢西瓜吃喽，呵呵。

所以现在看到西瓜就特么的有冲动想去把它砍两半，抱着吃。

是不是喜欢用勺子舀西瓜吃的人也特别多呢？
